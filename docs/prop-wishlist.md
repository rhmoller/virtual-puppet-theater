# 100 props a kid might request

A working list of props that children are likely to ask for during a
puppet theater session. Useful as:

- **Catalog inspiration** — anything a kid asks for repeatedly is a
  candidate to hand-author into `src/assets/catalog.ts` so it pops in
  instantly without an asset-gen round-trip.
- **Asset-generator test bench** — feed each entry into the asset
  lab (`/assetlab.html`) to see how well the LLM composes it from
  primitives.
- **Demo session prompts** — hand a kid this list during recording
  and let them pick whatever appeals.

Each prop is tagged with a likely mount: **head**, **eyes**, **neck**,
**hand_left** / **hand_right**, or **anchor** (a scene prop placed in
the theater, not on a puppet).
